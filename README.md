# CRM Laravel API

This repository contains a **REST API built with Laravel**, used as the backend for a CRM system.

It exposes secure endpoints consumed by multiple clients, including a Spring Boot application.

---

## Features

- JWT authentication (Laravel Passport)
- Role-based access control
- CSV multi-table import
- Export engine with metadata
- Business-level data validation

---

## Tech Stack

- Laravel
- PHP 8
- MySQL
- Laravel Passport

---

## Clients

- Spring Boot dashboard client
